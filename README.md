## アプリ名
Work,LiveShift
  
  
## アプリの概要
自分の性格を診断し、自分と合う企業をマッチングできる求人アプリです。  
ユーザーはアカウント作成の際にご自身の性格分析をしていただき、  
掲載されている企業様には企業の診断を行っていただきます。  
ユーザーの性格と企業の社風とをマッチさせ、  
自分に合ったお仕事を見つけていただくアプリケーションです。  
  
WebサーバーにNginx、APPサーバーにPumaを使用し、EC2へデプロイを行いました。  
DBサーバーには、RDSでMySQLを使用しています。  
CI/CDのためにCircleCIを導入し、テスト、デプロイを自動化しています。  
エラー、パフォーマンスの監視に、それぞれSentryとNewRelicを使用し、  
問題解決に素早く取り組めるようにしています。

## 目指した課題解決
就職や転職において、入社後に「思っていた企業と違う」や「本当にやりたかった仕事ではない」といった摩擦をなくし、
離職率が減らしたいと思い作成いたしました。
  

## 要件定義
 | 機能 | 目的 | 詳細 | ストーリー | 
 | ---|-----|------|---------- | 
 | ユーザー登録 | deviseを用いてユーザー管理機能 | トップページ右上にログイン、新規登録ボタンを表示させる。また、トップページの「ユーザーとして始める」を押すと新規登録画面へ遷移します。ログイン時はログアウトボタンを表示させる。| 企業にエントリーしたいユーザーは、ログインもしくは新規登録をする必要がある。 | 
 | ユーザーの性格診断 | ユーザーは新規登録の際に自身の性格を分析し、自分に合う企業を見つけることができる。 | ユーザー新規登録後、性格診断の画面へ遷移され性格診断を行う。性格診断終了後はご自身の診断結果を確認することができる。また、自身のユーザーページでも診断結果を表示させています。 | 転職や就職で失敗したくないユーザーは自分自身を客観的に理解することで、自分に合う仕事を見つけることができる。| 
 | ユーザーのプロフィール、経歴の追加、編集 | ユーザーは自分自身を企業の方によく知ってもらう。企業側はエントリーがあった人がどのような人なのかを知る必要があるため | ユーザー詳細画面にてプロフィールや、経歴の追加、修正、削除が行えます。自分以外の人には追加や修正はできないように設定しています。 | エントリーがあったユーザーがどのような人なのか企業側は面接の前に知ることで、ミスマッチを防ぐことができる。 | 
 | 掲載企業登録 | deviseを用いて企業管理機能 | トップページの「企業情報を掲載する」から企業登録を行うことができる。ウィザード形式の登録になっており、基本情報の他に会社の理念や求める人材の設定を行うことができます。企業登録が完了すると企業詳細画面に遷移されます。 | 求人を掲載したい企業は企業の登録を行う必要がある。 | 
 | 企業の社風診断 | 企業の社風、理念、雰囲気を診断し、自分の企業と合った人をマッチさせる。 | 企業登録後、企業詳細画面から社風診断が可能です。こちらは任意の診断になっています。 | 新たな人材を採用する際に、自分の企業と合う人とマッチさせることで離職率を下げることができる。 | 
 | 掲載企業一覧 | どのような企業が掲載されているのか誰でも確認することができる。 | トップページの「人気の企業を見る」から掲載企業の一覧を確認できる。ユーザ登録していない人やログイしていない人は人気順に、ログイン中のユーザーは自分と相性が良い企業順で並んでいます。 | 未ログインのユーザーは人気の企業を確認することができ、ログイン中のユーザーは性格診断を基に自分と相性の合う企業順に表示させることで自分に合った企業をすぐに見ることができる。 | 
 | エントリー機能 | ユーザーは働きたいと思った企業のエンタリーができる。 | 企業一覧から気になる企業を押すと、企業詳細に遷移されます。そこで良いと思った企業をエンタリーボタンからエントリーできます。エントリーを受け取った企業には通知がいき、企業側もエントリーを確認することができます。 | ユーザーは働きたい企業にエントリーをすることができる。 | 
 | お気に入り機能 | ユーザーは良いと思った企業をお気に入り登録ができる。お気に入りにした企業はいつでも確認することもできます。 | 企業一覧から気になる企業を押すと、企業詳細に遷移されます。そこで良いと思った企業をお気に入りボタンからお気に入りできます。もう一度お気に入りボタンを押すとお気に入りの解除ができます。お気に入りにした企業はいつでも確認することもできます。 | ユーザーは良いと思った企業にお気に入り登録をすることができる。 | 
 | 検索機能 | ユーザーが条件に合う企業を検索しやすくするため。 | 検索窓を用意。キーワード、働く場所、職種の３種類で検索をすることができます。 | ユーザーは検索窓から企業の条件を検索し、自分の条件に合う企業を絞り込むことができます。 | 
 | チャット機能 | エントリーをしてくれたユーザーと企業はチャットでやり取りすることができます。 | 企業はエントリーいただいたユーザーとチャットを始めることができます。未読のチャットメッセージががある場合は通知が表示されます。 | 企業はエントリーがあったユーザーの中で良いと思う人とチャットを始めることができます。 | 
 | 通知機能 | 企業とユーザーはチャットの新着メッセージやエントリーがあった際に通知を受け取る。 | 企業はエントリーと新着メッセージがあった際、ユーザーは新着メッセージがあった際に通知を受け取る。通知はヘッダーの通知ボタンに表示される。 | 企業もユーザーもメッセージやエントリーを見逃さないようにするため。 |
